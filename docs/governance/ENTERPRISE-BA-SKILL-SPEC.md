# Enterprise BA Skill Specification

Future skill: `enterprise-use-case-engineering`; implementation is Phase C1. Trigger for a new business capability/command/lifecycle transition/external integration/ISO flow/rail behaviour/user workflow/operational read model/aggregate candidate/material business semantic change.

It must produce or update Business Process, Business Use Case, System Use Case, Use-Case Slice, Example Mapping, business rules, alternatives/failures, quality scenarios, source traceability, architecture realization and epic/story mapping. Decision flow: find existing use case → update it, or create it → create/update slices → Example Mapping → source evidence → quality scenarios → architecture map → epic/story map. It must use `SOURCE-REGISTRY`, process catalogue, concept catalogue and UC2 templates; it must not infer participant behaviour or modify ADRs.
